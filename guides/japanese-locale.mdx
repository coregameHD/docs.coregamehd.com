<Tip>**Tip:** หากไม่ต้องการเปลี่ยน Locale สามารถใช้โปรแกรม [Locale Emulator](https://xupefei.github.io/Locale-Emulator) แทนได้</Tip>

### Windows 11

1. Start --> Settings

![Step1](https://i0.wp.com/geekrewind.com/wp-content/uploads/2021/07/windows-11-start-settings.webp?w=912&ssl=1)

2. Time & Language --> Language & region

![Step2](https://i0.wp.com/geekrewind.com/wp-content/uploads/2021/07/windows-11-display-language-change.webp?w=839&ssl=1)

3. Administrative language settings

![Step3](https://i0.wp.com/geekrewind.com/wp-content/uploads/2021/07/windows-administrative-language-settings.webp?w=849&ssl=1)

4. เลือกแท็บ Administrative จากนั้นเลื่อกปุ่ม Change system locale...

![Step4](https://i0.wp.com/geekrewind.com/wp-content/uploads/2021/07/windows-11-language-for-non-unicode-programs.webp?w=824&ssl=1)

5. บริเวณ Current system locale ให้เลือก Japan จากนั้นกดปุ่ม OK และ restart คอมพิวเตอร์หนึ่งครั้ง
